# Frontend Trabalho React

## Descrição

Aplicação web desenvolvida em React para a disciplina de Frontend.

O projeto implementa:

* Navegação entre páginas utilizando React Router;
* Integração com API REST;
* Listagem dinâmica de usuários;
* Busca de usuários em tempo real;
* Componentização com React;
* Estilização utilizando CSS externo;
* Responsividade básica.

## Tecnologias Utilizadas

* React
* Vite
* React Router DOM
* JavaScript
* CSS

## Estrutura do Projeto

* Home
* Cadastro
* Listagem

## Como Executar

```bash
npm install
npm run dev
```
